# Review do cliente - Old Brother V4

Fluxo sugerido para apresentação:

1. Entrar como Admin.
2. Mostrar Dashboard.
3. Abrir QR Codes e acessar Mesa 01.
4. Iniciar atendimento pelo QR Code.
5. Entrar em Mesas, criar dois pedidos separados para a mesma mesa.
6. Mostrar Central de Pedidos e Cozinha/KDS.
7. Enviar pedidos para pagamento.
8. Fechar a mesa no Caixa e mostrar faturamento.
9. Criar um Delivery e mostrar que ele vai para o Caixa antes do faturamento.
10. Mostrar Estoque, Produtos, Usuários e Relatórios.
